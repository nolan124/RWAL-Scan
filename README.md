# RWAL-Scan
Rich woman address list scanner --富婆地址列表扫描器 v1.0


Usage: python run.py -u <目标URL> -f <字典文件> [-t <线程数，这个参数可有可无>]，

示例：
python3 run.py -u http://example.com -f dict.txt

Options:
  -h, --help    show this help message and exit
  -u URL        要扫描的目标
  -f FILE_NAME  字典文件
  -t COUNT      线程数
